# Inbox

Use this folder as the first landing zone for teammate submissions when the correct section is not yet confirmed.

## Put Files Here First If

- the package spans multiple sections,
- the sender is unsure where it belongs,
- the drop mixes decks, screenshots, sheets, notes, and references,
- you want to preserve the untouched original package before sorting.

## Then Sort Into Section Folders

After intake review, move the files into the matching section:

- `02_sections/01_xd-epic-synergy/00_inbox/`
- `02_sections/02_reimagining-fortnite-for-china/00_inbox/`
- `02_sections/03_creator-engine/00_inbox/`
- `02_sections/04_solving-the-blocks/00_inbox/`
- `02_sections/05_the-offer/00_inbox/`
- `02_sections/06_appendix/00_inbox/`

## Recommended Naming

Use versioned names and do not overwrite earlier drops.

Examples:

- `fn_market_ownerdrop_v01.pptx`
- `fn_ops_assets_v02.zip`
- `fn_crosssection_notes_v01.docx`
