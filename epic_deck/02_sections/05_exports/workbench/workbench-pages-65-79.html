How the category is evolving

1.0

Core loop established

Built on standard survival-shooter conventions and the core extraction loop.

  • Map memory: spawn points, extraction points, POIs
  • Loot value systems: collectables, gear tiers, value gaps
  • Trading layer: sell loot, buy equipment
2.0

Differentiation through theme and systems

Keep the extraction core, then layer in stronger themes and lightweight gameplay twists.

  • World / theme variants around festivals or supernatural hooks
  • Micro-innovation in operators, traversal, and mode-specific systems
3.0

Toward co-op-heavy extraction

Shift from pure resource contest toward stronger PVE cooperation, with PVP becoming more situational.

Current: strong PVP / weak PVE

Direction: stronger PVE, lighter direct PVP conflict. Reference cases include ARC Raiders and other co-op-oriented extraction systems.
