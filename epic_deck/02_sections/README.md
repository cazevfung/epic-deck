# Sections Workspace

This folder is the main production area for the Epic deck.

## How To Use It

Each numbered section maps to one major chapter in the keynote.

Use the same workflow inside every section:

1. put owner submissions in `00_inbox/`,
2. move the chosen source material into `01_source-slides/`,
3. create one brief per page in `02_normalized-briefs/`,
4. record audit notes in `03_audit/`,
5. assign template and recipe in `04_template-mapping/`,
6. do working rebuilds in `05_rebuild/`,
7. keep local assets in `06_assets/`,
8. export reviewable outputs to `07_exports/`.

## Section Mapping

- `01_xd-epic-synergy`: XD, TapTap, capability proof, cases
- `02_reimagining-fortnite-for-china`: market, GTM, operations, TapTap advantage
- `03_creator-engine`: UEFN, UGC, creator ecosystem
- `04_solving-the-blocks`: technical integration and compliance path
- `05_the-offer`: commercial proposal
- `06_appendix`: supporting pages and overflow material
