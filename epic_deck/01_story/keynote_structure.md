# 一、Keynote结构

核心叙事逻辑是长期生态价值最大化：

- 心动有信心和能力触达和获取全量目标用户，不仅是 TapTap 的用户，也包括 WeGame 和 B 站的用户
- 心动擅长大 DAU 游戏和 UGC 游戏的长线运营和破圈，能够带来持续增长
- 在此基础上，不上渠道能够减少摩擦成本，提升用户体验，同时最大化双方以及创作者的收益
- 我们的商务条款是以上策略的结果，即短期需要保证产品及市场的投入和耐心，通过有竞争力的分成策略提升 Epic 的长期收益

核心信息：和 Epic 有着一样的不分成的理念，不仅是国内手游和跨端游戏最重要的分发平台，也是影响力最大的游戏社区，同时有着非常活跃的开发者生态。

## 2. Re-imagining Fortnite for China (市场洞察、产品定位、GTM、TapTap) @王飞

### 2.1 市场 @曹心鹤

Epic 非常擅长事件型营销，依靠各类跨界联动、流行文化的抓眼球营销事件带动社交传播，从而达到破圈的效果，营销事件常常和游戏内运营活动设计进行深度结合。

建议市场部分可以不用太务实，多开脑洞，让对方觉得性感很重要。

Fortnite 营销事件参考：

| 方法 | 代表案例 | 说明 |
| --- | --- | --- |
| 游戏内 live event | Travis Scott、Marshmello、Ariana、Remix Finale | 把发布、演唱会、赛季切换做成玩家共同参与的事件 |
| IP 赛季化 | Marvel Galactus、Star Wars | IP 进入地图、任务、Battle Pass、剧情和玩法，而不只是皮肤 |
| 悬念式版本营销 | Black Hole | 把更新、停服、版本切换包装成全球话题 |
| 怀旧召回 | Fortnite OG | 用经典地图和情绪记忆拉回老玩家 |
| 平台化发布 | The Big Bang | 用事件宣布 Fortnite 从单一游戏进入多体验生态 |
| 品牌岛屿 / UGC 商业化 | Nike Airphoria、Balenciaga | 让品牌成为可游玩的内容供给方 |

- 市场洞察及产品定位

- 重点营销事件和投入

## 2. 完整的 Fortnite China 上市规划和宣发策略 Go to Market Plan

- 基于 6 月确定合作，明年暑期上线的计划时间表（首爆，预热，测试，上线）
- 分阶段宣发的主基调，产品定位和核心卖点，发行最大的挑战和应对策略是什么
- 渠道分发的策略（Mobile 和 PC），渠道成功案例的经验分享
- 如何与国内的 IP 进行深度联动，怎么利用好 Fortnite 已有的 IP 矩阵？
- 外部流量平台的投放策略，二创生态的规划，创新的广告模式？
- 直播激励？主播资源？社区阵地？线下活动？

## 3. 运营的合作模式与具体分工

- 双方团队在发行事宜上的分工，长期高效的运作沟通模式
