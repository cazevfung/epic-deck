# 图片搜索任务 · Image Search Manifest
## 心动 × Epic Games · Fortnite 国服发行提案 Pitch Deck

---

## 背景

这是一份由心动网络（X.D. Network）向 Epic Games 提交的 Fortnite 中国发行合作提案的演示文稿（共17页）。你的任务是为文稿中每一个图片占位槽找到合适的图片，并提供可直接访问的链接。

---

## 你的任务

逐条处理下方【图片清单】中的每一个条目，为每条记录找到一张符合要求的图片，然后按照末尾【输出格式】的规范输出完整的 YAML 文件。

---

## 全局要求（适用于所有图片）

1. **分辨率**：所有图片必须 ≥ 1080p（即最短边 ≥ 1080px）。横图要求 ≥ 1920×1080，竖图要求 ≥ 1080×1920，方形图要求 ≥ 1080×1080。
2. **画面比例**：每条目注明了「首选比例」，但找不到首选比例时可接受横图或竖图，在 notes 字段说明即可。
3. **来源优先级**：官方网站 / 官方微博 / 官方微信公众号 / 游戏官方 Press Kit > 权威游戏媒体（游民星空、3DM、NGA、IGN CN 等）> 其他。
4. **可访问性**：URL 必须直接可访问，无需登录、无需付费、无需验证码。图片本身的直链（.jpg/.png/.webp）优先；如果只有页面链接也可以，在 notes 里注明。
5. **版权**：优先使用官方发布的宣传素材（Press Kit、官方公告配图）。不接受未授权粉丝二创图。
6. **如果找不到完全吻合的图片**：提供最接近的替代图，在 `status` 字段填 `"best_effort"`，在 `notes` 字段说明原因和与理想目标的差异。
