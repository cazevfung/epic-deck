# 战略定位

## 中国第一个跨界内容互动平台

在中国市场，传统游戏发行逻辑往往围绕单一玩法展开，强调竞技强度、商业化效率或短期用户获取。但《堡垒之夜》的核心优势并不局限于玩法层，而在于它同时具备“游戏产品”“内容容器”“社交场景”“品牌联动载体”四重属性。

这意味着，《堡垒之夜》在中国的机会，不是去争夺一个已经高度成熟的战术竞技存量赛道，而是开辟一个新的用户认知赛道，即以游戏为基础、以内容互动为核心、以创作者和跨界合作为增长引擎的长期平台生态。

中国用户已经成熟于接受线上娱乐内容的融合消费，用户并不只为“赢一局”付费或停留，他们也会为社交、表达、参与、围观和文化认同持续回流。因此，《堡垒之夜》在中国市场的最优解，不是被包装成单一竞技产品，而是被塑造成一个能够持续发生内容事件、承载创作者生态、连接品牌合作和线上娱乐体验的“新一代互动平台”。

# 首年市场投入

**$30,000,000**

首年我们建议投入总预算 3000 万美元，并以“首发爆发 + 长线生态”双轮驱动进行配置。
预算不是单纯买量，而是围绕《堡垒之夜》的平台型定位进行系统化布局。

## 投放分配

- **广告投放 (50%)**
  - 用于建立广泛市场认知、完成预约与下载转化，支撑重点版本放量。
- **品牌&联动 (20%)**
  - 用于打造破圈事件、IP联动、线下体验与首发社会化讨论。
- **UGC内容创作 (15%)**
  - 用于建立创作者内容供给、直播视频传播和 UEFN 创作者生态。
- **电竞赛事 (15%)**
  - 用于快速搭建中国本地赛事体系，提升核心玩家认同与长期生命力。

# 首发策略——$6,000,000

## 首发玩法——Zero Build

中国市场对于射击玩法的理解已经高度成熟，但“建造”机制可能在首发阶段提升新用户理解成本。因此，我们建议首发传播以 Zero Build 为核心表达，用最直接、最易理解的战术竞技体验承接更广泛的用户盘。

它能让用户以“爽快、直接、团队协作、观赏性强”的方式进入《堡垒之夜》，再在后续版本中逐步引导他们理解产品更完整的内容深度，包括建造玩法、创意地图、社交活动和 UGC 生态。

## IP联动+线下大事件

我们建议将首发传播与 IP 联动结合。中国有大量深受年轻一代欢迎的本土动画影视 IP，当这些 IP 角色通过《堡垒之夜》登上全球舞台，发挥出我们所擅长的联动创意后，中国玩家将产生前所未有的想象和期待。

《堡垒之夜》天然具备“内容事件化”的优势，线下城市主题快闪、沉浸式体验和活动跨界联动装置，能够把产品从“线上投放信息”转化成“可被拍摄、可被打卡、可被分享的公共事件”。

这类活动一方面能够帮助产品破圈，触达泛娱乐和年轻潮流人群；另一方面也能为社媒、短视频和直播平台持续提供素材，反哺线上热度。

# 创作者生态——$4,500,000

## 游戏直播与视频创作者激励

面向直播平台主播、短视频作者、攻略作者和娱乐内容创作者。目标是快速形成多层次内容矩阵。

由于SAC联盟计划对于中国用户比较陌生，所以会以内容激励+SAC的方式组合进行，期望最终将SAC计划在中国创作者群体中完成落地。

## 游戏内玩法创作者激励

这一部分面向更深层的生态参与者其核心目标不是短期传播，而是建立中国本地创作者供给，让《堡垒之夜》在中国市场从“一个官方游戏”进化为“一个持续生长的玩法平台”。

我们期望结合Creator Economy 2.0和岛内交易(In-Island Transactions)，激励中国本地创作者使用 UEFN，围绕本土文化和热点内容制作具有传播潜力的玩法内容。

我们将利用 TapTap 强大的开发者资源号召中国的专业游戏开发者们参与到《堡垒之夜》的内容创作中。

# 建立中国赛事体系——$4,500,000

赛事体系的作用，不只是服务少数高端玩家，而是帮助《堡垒之夜》在中国建立“可参与、可观看、可追随”的竞技叙事。

这将提升产品的核心用户粘性，也将持续为直播、视频和社交平台提供高质量内容。

## 赛事分类

### 大众赛事

- 面向全体玩家的公开赛事，赛事体系、奖励、赛制设置与国际服保持一致。
- 目标是让大众玩家都能感知并参与到《堡垒之夜》独特的竞技氛围，并有所收获。

### 职业赛事（FNCS 国服赛区）

- 国服独立的职业竞技周期，代表《堡垒之夜》国服最顶尖的竞技赛事，同时作为全球 FNCS 的中国赛区入口。
- 让国内的职业玩家征战全球竞技的重要平台。

### 平台与主播联赛

- 联合直播平台、头部主播和内容阵地发起中型赛事，兼顾观赏性与传播性。
- 目标是通过内容平台赛事提升《堡垒之夜》在核心圈层中的讨论热度。

# 全年投放计划——$15,000,000

首年总预算中的 50% 将投入线上投放，但投放节奏不是平均分配，而是围绕产品生命周期和关键节点进行分阶段推进。以确保每一轮预算都服务于不同阶段目标。

## 阶段规划

1. **预热期 ($3,000,000)**
   - 面向泛游戏用户、回归老玩家和内容平台受众，通过品牌视频、预约传播、KOL 首轮种草和核心媒体曝光，建立《堡垒之夜》回归中国市场的第一波认知。
2. **公测期 ($6,000,000)**
   - 是把认知快速转化为下载、激活与开服声量。
   - 同时配合主播直播、短视频传播和公测庆典事件，共同形成最大化市场引爆。
3. **日常版本 ($1,000,000)**
   - 保持市场存在感、覆盖节庆节点、支撑社群活动，并对高潜人群进行持续触达。
4. **版本更新期 ($4,500,000)**
   - 唤醒沉默用户重新放大内容势能。
   - 围绕新版本卖点、新活动、新联动和社区高光时刻进行再传播，让产品不陷入“上线即高点”的单峰曲线。
5. **日常版本 ($500,000)**
   - 保证全年热度不断档，为下一个大版本和下一年运营保留增长基础。
